# Routing Concepts

## What is Routing?

Routing is the process of forwarding data from one network to another using a router.

---

## What is a Default Gateway?

A default gateway is the router’s IP address used by a device to communicate with other networks.

---

## How do devices communicate across networks?

When a device wants to communicate with another network, it sends the packet to the default gateway. The router then forwards the packet to the destination network.
